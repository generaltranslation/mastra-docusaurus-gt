# Workflow.createRunAsync()

`.createRunAsync()` メソッドは新しいワークフロー実行インスタンスを作成し、指定した入力データでワークフローを実行できます。これは `Run` インスタンスを返す現行の API です。

:::note

メソッドを持つオブジェクトを返す旧式の `createRun()` メソッドについては、[Legacy Workflows](../../legacyWorkflows/createRun) セクションを参照してください。

:::

## 使い方の例

```typescript copy
await workflow.createRunAsync();
```

## 拡張的な使用例

```typescript showLineNumbers copy
const workflow = mastra.getWorkflow("workflow");

const run = await workflow.createRunAsync();

const result = await run.start({
  inputData: {
    value: 10,
  },
});
```
