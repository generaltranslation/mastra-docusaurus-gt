# Mastra.getAgentById()

`.getAgentById()` メソッドは、ID を指定してエージェントを取得します。このメソッドは、エージェントの ID を表す `string` 型のパラメータを 1 つ受け取ります。

## 使い方の例

```typescript copy
mastra.getAgentById("test-agent-123");
```
