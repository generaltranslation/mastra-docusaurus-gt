# Mastra.getDeployer()

`.getDeployer()` メソッドは、Mastra インスタンスで設定されたデプロイアインスタンスを取得するために使用します。

## 使い方の例

```typescript copy
mastra.getDeployer();
```


## パラメータ

このメソッドはパラメータを受け付けません。
