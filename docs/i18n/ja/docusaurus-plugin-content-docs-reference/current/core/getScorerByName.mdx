# getScorerByName()

`getScorerByName()` メソッドは、登録キーではなく `name` プロパティで検索してスコアラーを取得します。Mastra インスタンスでの登録名は分からないが表示名は分かっている場合に便利です。

## 使い方の例

```typescript
import { mastra } from "./mastra";

// name プロパティでスコアラーを取得
const relevancyScorer = mastra.getScorerByName("Answer Relevancy");

const weatherAgent = mastra.getAgent("weatherAgent");

// スコアラーを使用して AI 出力を評価
await weatherAgent.generate("ローマの天気はどうですか", {
  scorers: {
    answerRelevancy: {
      scorer: relevancyScorer,
    },
  },
});
```

## エラーハンドリング

このメソッドは次の場合に `MastraError` をスローします:

* 指定した名前のスコアラーが見つからない場合
* Mastra インスタンスにスコアラーが一つも登録されていない場合

```typescript
try {
  const scorer = mastra.getScorerByName("Non-existent Scorer");
} catch (error) {
  if (error.id === "MASTRA_GET_SCORER_BY_NAME_NOT_FOUND") {
    console.log("その名前のスコアラーが見つかりません");
  }
}
```
