Ollama は AI SDK で利用できます。Mastra でそのモデルを使うには、プロバイダー用パッケージをインストールしてください。

プロバイダー固有の詳細なドキュメントは、[AI SDK の Ollama プロバイダー ドキュメント](https://ai-sdk.dev/providers/community-providers/ollama)を参照してください。

このプロバイダーを Mastra のエージェントで使用する方法は、[Agent Overview ドキュメント](/en/docs/agents/overview)を参照してください。

## インストール

```bash npm2yarn copy
npm install ollama-ai-provider-v2
```
